# 📌 About Project

A beautiful and user-friendly **Simple Calculator** built using **C# Windows Forms**.

It performs basic arithmetic operations with a clean and modern interface.

# ✨ Features

✅ Addition

✅ Subtraction

✅ Multiplication

✅ Division

✅ Clear Button

✅ Exit Button

✅ Simple & Attractive UI

✅ Fast Calculation

✅ Beginner Friendly
